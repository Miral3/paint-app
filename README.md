# paint-app

## 과제
* canvas를 사용하여 그림판 만들기
* range를 이용하여 펜 너비 변경
* fill/paint 버튼을 이용하여 그리기, 색칠하기 모드 변경
* save 버튼을 이용하여 그림 저장
* color 버튼을 이용하여 색깔 변경

## 소요 시간 
3 ~ 4 시간
